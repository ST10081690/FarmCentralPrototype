ABOUT:
This application has the purpose of allowing
Farm Central employees to manage Farm Central's
farmers' details and viewing products. And it allows
Farm Central's farmers to create, read, update and 
delete products, to help them manage their farm's stock.

RUNNING (INSTRUCTIONS):

LOGIN PAGE --
Enter valid details into the textboxes and press the 
"Login" button to proceed.
--

HOME PAGE --
This page is your first destination after you logged in,
it displays a short description of the platform's purpose.
--

[FARMERS]--
ADD PRODUCT--
This page prompts you to create a new product by entering
specified details into the textboxes and clicling the 
"Add" button to save the details. 
(!) - The 'price' and 'quantity' inputs will not accept 
letters, only numbers should be entered. Please enter 
the details in order. Make sure to click the drop-down
list (for product type) and select one every time.
--

VIEW PRODUCTS--
This page allows you to see all the saved products that
belong to the farmer account that is logged in. You are
also able to update(edit) and delete items on this page.
--

[EMPLOYEES]--
ADD FARMER--
This page prompts you to create a new farmer by entering
specified details into the textboxes and clicling the 
"Add" button to save the details. 
(!)When adding a farmer USERNAME, ensure it includes
'farmer'.
--

VIEW FARMERS--
This page allows you to see all the saved farmers that
are present in the database. You are also able to update
(edit) and delete items on this page.
--

VIEW PRODUCTS--
This page allows you to view all products saved to the
database, and it allows you to filter products by the
farmer it belongs to, OR the farmer and the product type
OR the farmer and a date range.

Select an item from the drop-down lists to activate the 
filters. To change the filtering select a farmer and
then either a product type or a date range.

To clear all filters, click the 'Clear Filter' button.

(!) - When choosing a date range, select the start date,
then select the end date.
